# barrelr

## Installation

#### Install [`barrelr`](https://github.com/kylegilber/barrelr) from GitHub via [`pak`](https://github.com/r-lib/pak#arrow_down-installation):

``` r
if (!requireNamespace('pak', quietly = TRUE)){
  install.packages('pak')
}
pak::pak("kylegilber/barrelr")
```
